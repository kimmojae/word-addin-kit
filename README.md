# Word Add-in Kit

Word JavaScript API를 학습하고 테스트할 수 있는 개발 키트입니다.

## 기능

- 기본 API (텍스트, 단락, 서식)
- 문서 구조 (테이블, 헤더/푸터, 이미지)
- Content Controls (체크박스, 드롭다운, 날짜 선택기)
- 고급 기능 (각주, 문서 속성, 이벤트)

## 시작하기

```bash
# 설치
pnpm install

# 개발 서버 실행
pnpm dev
```

## 기술 스택

- Vue 3 + TypeScript
- Tailwind CSS
- Office.js
- Vite
